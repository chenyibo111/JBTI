# 📈 股票市场角色测试

> 20 道题，测测你是哪种股民

## 🎯 项目简介

这是一个模仿 MBTI 性格测试的股票市场角色测试工具，通过 20 道题目评估你在股票市场的行为特征，生成专属的股民类型报告。

## 📊 四个评估维度

| 维度 | 两极 | 含义 |
|------|------|------|
| **R** 风险偏好 | 🐢 保守派 ↔ 🚀 激进派 | 敢不敢梭哈 |
| **D** 决策风格 | 📊 数据党 ↔ 🔮 直觉派 | 看 K 线还是听消息 |
| **T** 持仓周期 | ⚡ 短线王 ↔ 🌲 长期主义 | 今天买明天卖 vs 传家宝 |
| **E** 情绪控制 | 😱 情绪化 ↔ 🤖 冷血机器 | 跌停哭 vs 加仓 |

## 🏆 16 种股民类型

- 📚 **价值投资宗师** - 你的操作风格像极了 2000 年的巴菲特
- 🧘 **佛系股东** - 嘴上说长期持有，跌停时比谁都难受
- 💼 **稳健交易员** - 赚小钱的高手，亏大钱？不存在的
- 😰 **焦虑套利者** - 每天都在卖飞和套牢之间反复横跳
- 🎰 **理性赌徒** - 敢梭哈还能睡得着，你是真·大心脏
- 🎢 **韭菜中的战斗机** - 你的交易记录可以拍成纪录片《如何快速亏钱》
- ⚡ **短线收割机** - 今天买明天卖，券商最爱你的手续费
- 🔥 **追涨杀跌王** - 高位接盘·低位割肉·循环往复·永不言弃
- 🦅 **独狼猎手** - 不听消息不跟风，有自己的交易节奏
- 🐿️ **松鼠型囤积者** - 听消息买股，然后囤着当传家宝
- 📰 **消息灵通人士** - 群里第一个转发利好消息的，也是第一个跑的
- 🌪️ **风中凌乱者** - 今天信这个明天信那个，账户像坐过山车
- 🤖 **人形量化机器** - 没有感情的交易执行器
- 📉 **数据型韭菜** - 分析得头头是道，操作得一塌糊涂
- 🎯 **精准狙击手** - 研究透了再下手，下手了就跑，绝不留恋
- 💣 **爆仓预备役** - 杠杆加满·全仓进出·心态爆炸·周而复始

## 🚀 使用方法

### 本地运行

直接用浏览器打开 `index.html` 即可：

```bash
# 或者用 Python 起个简单服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```

### 部署到 GitHub Pages

1. Fork 本仓库
2. 进入 Settings → Pages
3. 选择 `main` 分支和 `/ (root)` 目录
4. 访问 `https://你的用户名.github.io/stock-test/`

## 📁 项目结构

```
stock-test/
├── index.html          # 主页面
├── style.css           # 样式
├── script.js           # 题目数据 + 计分逻辑
└── README.md           # 说明文档
```

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- 无依赖库
- 响应式设计（支持手机/平板/桌面）

## 📝 题目设计

- 20 道题目，每题 3 个选项
- 每个维度 5 道题
- 计分范围：-5 到 +5
- 根据正负倾向匹配 16 种类型

## 🎨 自定义

### 修改题目

编辑 `script.js` 中的 `questions` 数组。

### 修改结果类型

编辑 `script.js` 中的 `resultTypes` 对象。

### 修改样式

编辑 `style.css`，主色调为渐变紫色 `#667eea` → `#764ba2`。
